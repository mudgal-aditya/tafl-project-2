# CFG Simplifier Web App

A web-based tool to simplify **Context-Free Grammars (CFGs)** step by step, with full visual trace and explanations.

---

## Overview

This project demonstrates the complete process of CFG simplification:

1. **Remove Useless Symbols**
2. **Remove Null (ε) Productions**
3. **Remove Unit Productions**

Each transformation is applied in the correct order and displayed with:
- Before vs After comparison
- Step-by-step reasoning
- Visual trace of changes

---

## Features

- Step-by-step CFG simplification
- Full trace panel with explanations
- Interactive UI with clean visualization
- Multiple example grammars
- “Run All Steps” execution
- Theory section explaining concepts and algorithms

---

## Tech Stack

- HTML (structure)
- CSS (UI styling)
- JavaScript (logic + transformations)

No frameworks or dependencies.
