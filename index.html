Theory & Concepts

Formal definitions and algorithms behind each grammar simplification step.

What is a Context-Free Grammar?

A CFG is a 4-tuple G = (V, Σ, R, S) where V is a finite set of variables, Σ is a finite set of terminals (disjoint from V), R is a set of production rules A → α where A ∈ V and α ∈ (V ∪ Σ)*, and S ∈ V is the start symbol.

A grammar is in simplified form when it contains no useless symbols, no null (ε) productions (except possibly S → ε if ε is in the language), and no unit productions. Simplification is a prerequisite for converting grammars to normal forms like CNF or GNF.

The order of simplification always matters: useless symbols first, then null productions, then unit productions. Changing the order can leave the grammar in an incorrect state.

1 Removing Useless Symbols

A symbol X is useless if it is non-generating or unreachable. Two sub-passes are required and must run in order:

  • Generating: X can derive a string of only terminals.
  • Reachable: X can be derived from S through some sequence of productions.
Pass 1 — Generating: generating = {} repeat until no change: for each rule A → α: if all symbols in α are terminals or already generating: add A to generating remove non-generating symbols Pass 2 — Reachable: reachable = {S} BFS/DFS from S through productions remove unreachable symbols

2 Removing Null Productions

A null production is A → ε. To eliminate them without changing the language:

  • Find all nullable variables (those that derive ε directly or transitively).
  • For each rule, add all combinations where nullable variables are optionally dropped.
  • Remove all ε-rules from the grammar.
nullable = {} repeat until no change: for each rule A → α: if α = ε, or every symbol in α is already nullable: add A to nullable For each rule A → X₁X₂…Xₙ: generate all 2ⁿ combinations where nullable Xᵢ may be dropped (excluding the empty combination)

3 Removing Unit Productions

A unit production is A → B where B is a single variable. Removed via unit-pair closure:

  • (A, B) is a unit pair if A ⟹* B using only unit rules.
  • For each unit pair (A, B), add all non-unit productions of B directly to A.
  • Remove all unit productions from the grammar.
unit_pairs(A) = {A} for each A repeat until no change: for each rule A → B (unit rule): unit_pairs(A) ∪= unit_pairs(B) For each (A, B) in unit_pairs: for each non-unit rule B → α: add A → α to grammar Remove all A → B rules

Why Order Matters

Correct order: Useless → Null → Unit

  • Removing null productions can create new unit productions (e.g. A → BC where B is nullable produces A → C, a unit production).
  • Removing unit productions first can leave ε-rules that later expand into new unit productions, undoing the work.
  • Removing useless symbols first reduces the grammar, preventing unnecessary expansion of nullable combinations during step 2.

Concrete example of order dependency:

Original: S → AB, A → aA | ε, B → b After null removal (step 2): S → AB | B ← new unit prod! A → aA B → b Then unit removal (step 3) handles S → B by substituting: S → b
